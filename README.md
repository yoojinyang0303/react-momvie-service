# Nomad Coders - ReactJS로 영화 웹 서비스 만들기

## 강의 내용
영화 api로부터 불러온 영화 정보를 토렌트 다운로드 횟수 기준으로 인기 영화 정렬된 간단한 영화소개 웹 사이트 클론코딩
<br/><br/>
